LoL Friend Manager is a Windows app made to help you manage your League of Legends friends list in a simple, fast, and safe way. It is distributed as a single `.exe`, so there is no installer to go through.

## Download

1. Download the `.exe` file from this repository.
2. If there is more than one build available, use the newest one.
3. Double-click the file to launch it.
4. If Windows shows a security prompt, choose the option to keep or run the file only if you trust the source.

Main file: [LoLFriendManager.exe](./LoLFriendManager.exe)

## What the app can do

- Connect to your LoL client and load your friends list.
- Refresh the list when needed.
- Filter friends by folder.
- Select all visible items at once.
- Remove selected friends in bulk.
- Remove everyone inside the current folder.
- Remove everyone outside a chosen folder.
- Move selected friends to another folder.
- Use a dry run mode to preview what would happen before applying changes.
- Adjust the delay between removals.

## How to use

1. Open League of Legends.
2. Run `LoLFriendManager.exe`.
3. Wait for the friends list to load.
4. Pick a folder or select the friends you want to manage.
5. Choose the action you want to apply.

The app always shows confirmation prompts for sensitive actions, so you can review everything before making a final decision.

## FAQ

### Can I use it with Valorant?

No. LoL Friend Manager is only for the League of Legends friends list.

### Do I need to install anything?

No. Just download the `.exe` and open it.

### Does it work on Mac or Linux?

No. This version is for Windows only.

### Does it ban accounts in LoL?

There are no known reports of bans coming from LoL Friend Manager so far. The app is designed for regular friend-list management, includes confirmations, and offers a dry run mode so you can preview changes before applying them.

### Is bulk removal safe?

The app adds confirmation steps before important actions, but removal is still permanent. Always review the selection before confirming.

### Can I try it before making changes?

Yes. Use dry run mode to see what would happen without changing anything.

### Why is the list not loading?

Open the LoL client first, then try again.

## Notes

- For the smoothest experience, always use the latest `.exe` build.
- If the client was closed or restarted, refresh the list inside the app.
- If your browser, Windows, or antivirus marks the file as suspicious, this can happen with unsigned `.exe` downloads. Only continue if you downloaded it from this official repository.
